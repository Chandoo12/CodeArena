# CodeArena - Collaborative Coding Interview & Contest Platform

A full-stack web application inspired by LeetCode and HackerRank that allows users to solve coding problems, participate in contests, and conduct real-time coding interviews in a shared editor.

## 🎯 Features

### Core Features
- **Authentication**: User registration & login with JWT
- **User Dashboard**: Profile, stats, submission history
- **Problem Management**: Browse, search, and filter coding problems
- **Online Judge**: Code execution with test cases and verdicts
- **Contests**: Create and participate in timed contests
- **Real-Time Interview**: Shared code editor for interviews

### Tech Stack
- **Frontend**: React.js + TypeScript + Tailwind CSS + Monaco Editor
- **Backend**: Node.js + Express.js + Socket.io
- **Database**: PostgreSQL
- **Deployment**: Vercel (frontend) + Render (backend)

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- PostgreSQL
- Git

### Installation

1. **Clone repository**
   ```bash
   git clone <your-repo-url>
   cd CodeArena
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. **Setup Backend**
   ```bash
   cd backend
   npm install
   npm run dev
   ```
